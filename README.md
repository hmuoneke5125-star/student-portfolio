# Student Portfolio

A simple portfolio website to showcase projects and personal information.

## What's Inside

- **HTML Files**: Main pages for the site (home, about, projects, contact, planner)
- **CSS**: Styling for the whole site
- **JavaScript**: Interactive features and functionality
- **Images**: Project and profile pictures

## Pages

- `index.html` - Home page
- `about.html` - About me
- `projects.html` - My projects
- `contact.html` - Get in touch
- `planner.html` - Project planner tool

## How to Use

Just open `index.html` in your browser. Everything else should load from there.

## Files

- `css/style.css` - All the styling
- `js/script.js` - JavaScript for interactive stuff
- `images/` - Photos and project images

That's it. Simple and straightforward.
